Exercises¶

  1. Open Jupyter Lab:

    • Windows: Open Anaconda Prompt and type jupyter lab and hit enter

    • Mac: Open a terminal window and type jupyter lab and hit enter

  2. Use the left sidebar’s file browser to

    1. Nav to my textbook folder to copy today’s exercises

    2. Nav to my Class Notes folder, create a folder called “Exercises”

    3. Go into the Exercises folder you just made and paste the exercises ipynb file there

    4. Open your “Module 1 Notes” file and the exercise file from inside your notes folder

  3. Arrange the notes and exercise file as two halves of the work area so you can see both at the same time.

  4. Writing python

    • go down to the “Jupyterlab Notes” part of the file

    • under it, there is “put your notes here”. Replace that with print("Hello world!") (don’t forget both quote marks or parentheses)

    • Do you notice the syntax coloring?

    • hit the “play” button to run the cell

    • Do you see the number to the left of the output? That helps you track the order in which each cell was run.

    • Change that line so it says print("Hello world!") # this is a comment

    • Notice the coloring of what you just added

    • The hashtags/pound signs are the “comment signal”. They tell Python to ignore everything to the right of it on that line, so that you can write notes to anyone looking at the code.

    • this time, to run the code, hit ctrl + enter on your keyboard

    • notice how next to the output it used to say “[1]” but now says “[2]”

  5. Modal editing system

    1. Because jupyterlab has parts that are code and parts that are markdown text, it uses a “modal” editing system so that you can either (A) edit code and text within the cells you’re writing which is called “editing mode” or (B) modify the cells themselves (e.g. add a new cell, change cell from markdown to code, …) using command mode

      • Switch into edit mode by hitting Enter or double clicking in the cell.

      • Switch into command mode by hitting ESC

    2. With your cursor in the “hello world” cell, hit ESC. Notice how the coloring on the screen changes. Now hit ENTER to go back to edit mode.

    3. add a new cell below it by clicking the plus button on near the top of work area.

    4. Add a second new cell below by keyboard shortcut. The shortcuts are in chapter 1.6.3. So, click ESC to switch to command mode, then B to insert a cell B elow.

  6. Writing markdown (more at chapter 1.5)

    • Switch the code cell you just inserted cell to markdown (via the markdown/code drop down box) and write a silly sentence. The click CTRL + ENTER. See?

    • Now delete the cell with that silly sentence: ESC to go to command mode and then D,D (hit the “d” button twice).

  7. Learning more

    1. Read Chapters 1.5 and 1.6. Take notes in the notes file!

    2. I’d copy the shortcuts into your notes file, (as markdown, under the “Juypyterlab note” header). Even better: practice them!

Short summary

  1. You’ve added and deleted cells.

  2. You can change a code cell to markdown and back.

  3. You’ve used some shortcuts.

Machine learning¶

We will spend the rest of class learning

  1. how to build a prediction model

  2. how to optimize a prediction model

  3. making that work with real world (messy!) data

References¶

  1. Chap 7 explains many of the WHYS behind why we are doing what we are doing and introduces the new vocabulary you’ll need to learn

  2. Chap 8 gives concrete code and pseudo code so you can get to work

  3. sklearn is the python package that most ML is built on (and other packages use identical or similar syntax).

    • Must read: Getting started, a few short examples

    • BOOKMARK NOW: You will visit its user guide frequently, it’s really good!

    • Glossary of terms

    • The pipeline page

    • Preprocessing options

    • Dealing with missing values

    • Do you have too many variables? Two options: Supervised feature selection or unsupervised

    • The “menu” of supervised learning models - go here if you are want to learn more about a model we run in class or see what’s possible

    • Info on cross validation

    • How to optimize a model

    • What scoring functions can I set a model to optimize?

  4. EDA. Remember: ONLY do EDA on the training sample (never on the holdout sample)

    • ydata-profiling is great with two caveats: can be slow, generic insights

    • dabl is great - will suggest cleaning options and help see likely drivers of good models, and will even BUILD a starter model for you. But: BLACK BOX! And Caveat: It doesn’t do CV.
